# 🔗 工作流模板

> 三大模块组合使用，覆盖常见创作场景

---

## 模板A：看到好视频 → 做出同款

```
场景：你刷到一段穿搭展示视频，想复制风格做自己的服装视频

步骤：
1. 下载视频 → yt-dlp "URL" -o "参考视频.mp4"
2. 分析分镜 → L1/01 参考视频分析
               逐镜头提取景别/运镜/灯光/动作/节奏
3. 生成提示词 → L1/02 提示词工程
               把分析结果写成即梦@引用格式
4. 精调控制 → L1/03 导演级控制
               微调镜头运动(seedance-camera)
               调整光线方案(seedance-lighting)
5. 生成视频 → 即梦/可灵粘贴提示词
```

## 模板B：多套衣服换装展示

```
场景：你有N件衣服 + 1个模特 + 1个场景，想做换装合集

平台选择：
  多套衣服 + 简单动作（转身/展示） → 即梦 Seedance 2.0
  单套衣服 + 复杂动作（走秀/精准复刻） → 可灵 Kling 3.0

即梦路线：
  1. @人物 + @每件衣服 → 多参考图，分段生成
  2. 每段提示词走 L1/02 模板
  3. L1/03 加镜头运动一致性
  4. N段视频 → 剪映/PR拼接为合集

可灵路线：
  1. 虚拟试穿API → 人物上身图
  2. 上传动作模板/参考视频
  3. 角色参考保持人物一致
  4. 每件衣服单独生成 → 拼接
```

## 模板C：产品展示视频

```
场景：一个产品（可乐/鞋子/香水）需要展示视频

1. 如果没参考视频：
   → L1/02 直接写提示词（场景化策略-电商模板）
   → L1/03 加入镜头语言（360度旋转/微距特写）

2. 如果有参考视频（竞品广告）：
   → L1/01 拆解竞品分镜结构
   → 替换产品为你的产品
   → L1/02 重新包装提示词
```

## 模板D：从零到完整短片

```
场景：有剧情想法，想做一段短视频

步骤：
1. 脚本分镜 → 手动写分镜头表（几个镜头、每个多长）
2. 片段生成 → 多镜头序列工作流
               L1/03 seedance-sequence 规划全线
               每段独立生成，记录终点画面
3. 拼接组合 → 按分镜顺序拼接各段视频
4. 音频补充 → 后期加背景音乐/配音
```

## 模板E：视频延长（超15秒）

```
1. 先正常生成第1段15秒视频
2. 把生成的视频上传为 @视频1
3. 选择"延长"模式
4. L1/02 写延长提示词：接@视频1最后一帧继续
5. L1/03 检查运镜方向/动作节奏是否一致
6. 可反复延长，每段+15秒
```
