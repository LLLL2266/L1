## @引用系统

Seedance 2.0 支持多模态参考，使用 `@` 语法：

| 引用 | 说明 | 限制 |
|------|------|------|
| `@图片1` ~ `@图片9` | 角色/场景/产品参考图 | 9张, 每张<30MB |
| `@视频1` ~ `@视频3` | 运镜/动作参考视频 | 3段, 每段2-15s, <50MB |
| `@音频1` ~ `@音频3` | 背景音乐/音色参考 | 3个, 每段≤15s, <15MB |

**使用示例：**
```text
@图片1中的人物身穿@图片2中的红色连衣裙，站在@图片3的场景中，
缓缓转身展示服装细节，镜头从全景缓慢推至半身特写。
```

**⚠️ 平台限制：** 不支持上传含写实真人脸部的素材（会被拦截）。如需真人角色，建议用可灵。

## 提示词结构（Director Formula）

```
[主体] + [动作] + [场景] + [镜头] + [光线/风格] + [音频] + [约束]
```

| 槽位 | 用途 | 示例 |
|------|------|------|
| 主体 | 画面锚点 | 一位年轻女性穿着白色连衣裙 |
| 动作 | 可见变化 | 从左侧缓步走入，轻轻转身展示裙摆 |
| 场景 | 环境描述 | 简约白色摄影棚，自然光 |
| 镜头 | 单一运镜+终点 | 镜头从全景缓慢推至半身 |
| 光线/风格 | 物理光线+视觉语言 | 柔和顶光，干净商业摄影风格 |
| 音频 | 环境音/对白/音效 | 轻柔背景音乐，裙摆飘动声 |
| 约束 | 保留/排除 | 保持人物面部一致，服装细节清晰 |

## 时间戳分镜法（13-15秒视频）

用时间戳精确控制每段：

```text
15秒服装展示视频，0-3秒：@图片1人物从画面右侧走入，
4-8秒：人物在场景中央缓缓转身，双手轻抚裙摆展示细节，
9-12秒：镜头缓慢推至半身特写，人物微笑看向镜头，
13-15秒：画面定格在人物侧面优雅姿态。
```

---

## 超长视频拼接策略（>15秒）

```text
第1段（0-15秒）：正常生成 → 视频结尾定格
                   ↓
第2段（15-30秒）：将第1段视频上传为 @视频1 → 选择"延长"模式
                   ↓
第3段（30-45秒）：接第2段结尾 → 继续延长...
```

## 场景化提示词策略

### 电商/服装展示
```text
@图片1人物穿着展示@图片2服装，站在简约白色摄影棚中，
缓慢360度转身展示服装细节，柔和顶光，
干净商业摄影风格，保持服装版型和颜色准确。
```

### 产品广告
```text
@图片1产品在黑色亚克力台面上缓慢旋转，
表面水珠凝结滑落，5秒内从全景推到微距特写，
动态商业摄影，暖色侧光，背景深邃虚化。
```

### 短剧/对话
```text
@图片1角色坐在咖啡厅靠窗位置，端起咖啡杯轻啜一口，
然后缓缓放下杯子，看向窗外若有所思，
窗外光影在脸上流动，35mm电影镜头质感。
```

## 禁用词

避免以下词，会让输出变油：
```
cinematic, professional, stunning, 8k, studio, perfect,
史诗级, 震撼, 极致, 完美, 顶级, 电影级(作为修饰)
```

改用具体描述：
```
❌ "电影级视觉效果"
✅ "浅景深，暖色温，35mm镜头焦段"
```

## 即梦 vs 可灵 提示词差异

| | 即梦 Seedance | 可灵 Kling |
|--|--------------|-----------|
| 风格 | 自然中文描述 | 中文提示词效果更好 |
| 镜头控制 | 支持详细镜头语言 | 支持基础镜头描述 |
| @引用 | @图片/@视频/@音频 | 角色参考/元素参考 |
| 动作控制 | 靠提示词描述 | 支持上传动作模板 |
| 最佳实践 | 写好参考帧再补动作 | 首帧质量决定上限 |
| 真人脸 | ❌ 不支持 | ✅ 支持 |
